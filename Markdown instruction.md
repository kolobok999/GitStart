# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходимо обрамить его звездочками (*). *Курсив*

Чтобы выделить текст полужирным, необходимо обрамить его двойными звёздочками (**). **Полужирный**
